# 🧠 memoria-squadra/ — i quaderni dei 20 senior

Un quaderno per senior (`<nome>.md`), **append-only**. È la memoria lunga del ruolo: ogni lavoro
importante lascia una **riga ESITO**.

Formato riga: `AAAA-MM-GG · contesto · cosa ha funzionato o no · numero · lezione · #tag`

Regole:
- Non riscrivere le righe vecchie. Si aggiunge in fondo.
- Ogni settimana si può RIASSUMERE/POTARE per tenerli piccoli (la storia resta nei briefing/DECISIONI).
- Il rituale d'inizio di ogni senior comincia leggendo il proprio quaderno.
