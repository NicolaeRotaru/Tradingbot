# 📦 consegne/ — gli artefatti finiti di TradeDesk OS

Qui i senior depositano il **lavoro fatto** (non descrizioni): report, backtest, dossier, proposte,
audit. È la prova del doer-mode.

## Convenzioni
- Nome file: `AAAA-MM-GG-<argomento>.md` (o `.csv`/`.png` per dati/grafici).
- Gli **audit profondi** (radiografia, audit strategia) vanno in `consegne/audit/`.
- I **report del diario** li genera `python cervello/diario.py report` (in `consegne/`).
- Ogni artefatto importante cita **dati reali** e dichiara l'effetto sul KPI del reparto.

## Sottocartelle
- `audit/` — report degli audit profondi (per gravità: 🔴 bloccanti / 🟠 seri / 🟡 minori).
